## 1. Configuración del sensor de etiquetas en la i7100

**Artículo oficial:**  
[i7100 - Label Sensor Setting on the Printer](https://support.bradyid.com/s/article/i7100-Top-Reflect-Sensor-Setting)

### Descripción

Este artículo explica dónde configurar el tipo de sensor de etiquetas desde la pantalla táctil de la impresora i7100. La configuración se encuentra dentro del menú **Setup** de la impresora. La i7100 permite seleccionar diferentes configuraciones de sensor, como **Gap Sensor**, **Bottom Reflect** y **Top Reflect**, dependiendo del tipo de material utilizado. 【1-f72d9a】

### Ruta de configuración

En la pantalla de la impresora:

```text
Gear Icon > Setup > Labels > Label Sensor
